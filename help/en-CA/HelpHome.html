Welcome to Catheedral!
The Catheedral Home page has four main components: Time, Climate, Weather, and People. In addition, there are Navigation Controls for moving left and right between pages, and four Main Buttons, one in each corner.
Time
This displays the current time for your locale. Additional information, such as sunrise/sunset and the current phase of the moon, may also be available if Home Assistant has been configured to provide this information to Catheedral. There are as many as five rings in this panel.

  • Seconds shows the current second, much like a second hand on a traditional clock.
  • Minutes shows the current minute, much like the minutes hand on a traditional clock.
  • Hours shows the current hour, but unlike a traditional clock, this is calibrated for 24 hours.
  • Dawn/Dusk shows the portion of the day between Dawn and dusk, using the same 24-hour calibration as Hours.
  • Sunrise/Sunset shows the portion of the day between Sunrise and Sunset, using the same 24-hour calibration as Hours.
In addition, the length of time between Sunrise/Sunset is displayed as Daylight and the the length of time between Dawn/Dusk is displayed as Twilight.
Navigation Controls
At the very left and very right of the Catheedral interface are the Navigation Controls that let you move back and forth between related pages.
Main Buttons
There are four buttons, one in each corner, that are always visible within Catheedral. These are referred to as Main Buttons.

The Help button in the top left corner will present you with information related to the page you're currently viewing. You can also view other Help pages by using the Navigation Controls while viewing any Help page.

The Change button in the top right corner provides access to additional settings for the page. In some instances, this will change to a Refresh button that will allow you to reload that particular page.

The Home button in the bottom right corner will bring you back to the Catheedral Home page. If you're already on the Catheedral Home page, this changes to a Connected button, which can be used to reset the connection between Catheedral and your Home Assistant server. If the connection between Catheedral and your Home Assistant server is lost, this will change to a Disconnected button.

The Configuration button in the bottom left corner brings up additional pages that allow you to make changes to how Catheedral works. If you're already on the Configuration page, this button will instead change to a Power Off button that will allow you to exit the Catheedral application completely.
